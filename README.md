## KM Finanzmanagement

KM Finanzmanagement ist eine angepasste Version der Open-Source-Buchhaltungssoftware Frappe Books, speziell entwickelt für die Bedürfnisse kleiner Unternehmen und Freelancer. Mit einer intuitiven Benutzeroberfläche und erweiterten Funktionen für Ausgaben- und Einnahmenmanagement bietet es eine umfassende Lösung für die moderne Finanzverwaltung.

### Motivation

KM Finanzmanagement erweitert die solide Basis von Frappe Books um spezialisierte Funktionen für das moderne Ausgaben- und Einnahmenmanagement. Es bietet eine intuitive, deutschsprachige Lösung, die Einfachheit mit wesentlichen Funktionen kombiniert und Unternehmen dabei unterstützt, ihre Finanzen effektiv zu verwalten.

### Hauptfunktionen

- **Dashboard**: Bietet einen Überblick über wichtige Finanzdaten und Leistungskennzahlen.
- **Ausgabenmanagement**
  - **Einmalige Ausgaben**: Erfassung und Verwaltung einmaliger Geschäftsausgaben mit Kategorisierung, Steuerbeträgen und Anhängen.
  - **Wiederkehrende Ausgaben**: Automatisierte Verfolgung regelmäßiger Ausgaben mit PDF-Upload-Funktionalität für Rechnungen.
  - **Ausgaben-Dashboard**: Zentrale Übersicht aller Ausgaben mit Filteroptionen und Auswertungen.
- **Einnahmenmanagement**
  - **Einmalige Einnahmen**: Erfassung von Barzahlungen, Trinkgeldern, Nachzahlungen und anderen einmaligen Einnahmen.
- **Zahlungsverifizierung**
  - **Intelligenter Abgleich**: Automatischer Abgleich von Banktransaktionen mit erfassten Ausgaben.
  - **Gespeicherte Verifizierungen**: Verwaltung und Nachverfolgung von Verifizierungsergebnissen.
- **Kundenverwaltung**
  - **Abonnement-Kunden**: Verwaltung von Kunden mit wiederkehrenden Zahlungen und Abonnements.
- **Offline-Funktionalität**: Ermöglicht das Arbeiten ohne Internetverbindung mit späterer Synchronisation.
- **Doppelte Buchführung**: Gewährleistet genaue Finanzverfolgung durch Aufzeichnung jeder Transaktion in zwei Konten.
- **Berichte und Auswertungen**
  - **Hauptbuch**: Zentraler Datensatz aller Finanztransaktionen.
  - **Gewinn- und Verlustrechnung**: Zusammenfassung von Einnahmen, Kosten und Ausgaben.
  - **Bilanz**: Zeigt Vermögenswerte, Verbindlichkeiten und Eigenkapital eines Unternehmens.
  - **Probebilanz**: Überprüft die Genauigkeit der Buchführungsunterlagen.
    <br/>

### Technische Basis

- **Vue.js**: KM Finanzmanagement nutzt Vue.js für das Frontend und ermöglicht eine reaktive, komponentenbasierte Benutzeroberfläche mit nahtlosen Interaktionen und dynamischen Updates.

- **Electron**: Electron wird verwendet, um KM Finanzmanagement als eigenständige Desktop-Anwendung zu verpacken, die offline funktioniert und eine native Benutzererfahrung unter Windows, macOS und Linux bietet.

- **SQLite**: Alle Finanzdaten, Transaktionen und Konfigurationen werden sicher in einer SQLite-Datei auf dem Computer des Benutzers gespeichert.

## Installation und Setup

### Entwicklungsumgebung

### Voraussetzungen

Für die Entwicklungsumgebung benötigen Sie Node.js `v20.18.1` und npm. Wir empfehlen die Verwendung von [nvm](https://github.com/nvm-sh/nvm#installing-and-updating).

Zusätzlich müssen Sie [yarn](https://classic.yarnpkg.com/lang/en/docs/install/#mac-stable) installieren.

### Klonen und Ausführen

Nach der Installation der Voraussetzungen können Sie folgende Befehle ausführen, um KM Finanzmanagement für die Entwicklung einzurichten:

```bash
# Repository klonen
git clone https://github.com/kuhlertfl/Finanzmanagament.git

# Verzeichnis wechseln
cd Finanzmanagament

# Abhängigkeiten installieren
yarn
```

Um KM Finanzmanagement im Entwicklungsmodus auszuführen (mit Hot Reload, etc.):

```bash
# Electron-App starten
yarn dev
```

**Hinweis: Erster Start**

Beim Ausführen von `yarn dev` startet Electron sofort, aber die Benutzeroberfläche benötigt einige Sekunden zum Rendern. Dies liegt daran, wie der Entwicklungsmodus funktioniert - jede Datei wird einzeln vom Dev-Server bereitgestellt.

**Hinweis: Electron Main Process debuggen**

Im Entwicklungsmodus läuft Electron mit dem `--inspect` Flag, was es einem externen Debugger ermöglicht, sich mit Port 5858 zu verbinden. Sie können Chrome dafür verwenden, indem Sie `chrome://inspect` besuchen, während KM Finanzmanagement im Dev-Modus läuft.

#### Build

Um KM Finanzmanagement zu erstellen und einen Installer zu generieren:

```bash
# Electron-App erstellen
yarn build
```

**Hinweis: Build-Ziel**
Standardmäßig erstellt der obige Befehl für das Betriebssystem und die Architektur Ihres Computers. Um für andere Umgebungen zu erstellen, überprüfen Sie den _Building_ Abschnitt unter [electron.build/cli](https://www.electron.build/cli).

Für Linux können Sie das `--linux` Flag verwenden: `yarn build --linux`.
